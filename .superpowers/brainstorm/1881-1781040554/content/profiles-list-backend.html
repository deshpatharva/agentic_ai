New Backend Endpoints

Method + Route Purpose
POST /profile/parse Raw text → structured sections JSON (LLM)
POST /profile/ai-interview/message One turn of the AI interview (stateless, history in body)
POST /profile/ai-interview/finish Synthesize full conversation → sections JSON
POST /profiles Create profile (saves sections JSON + label)
GET /profiles List user's profiles
GET /profiles/:id Get profile detail (full sections)
PUT /profiles/:id Update label or sections (bullet edits)
DELETE /profiles/:id Delete profile
POST /profile/match Given JD text → ranked profile list with match %
POST /jd/scrape URL → JD text (generic scraper, 24h cache)
PUT /run-pipeline Updated to accept profile_id (in addition to job_id)
